# DCF_Automatizado
Pipeline automatizado de valuation por Fluxo de Caixa Descontado (DCF). O algoritmo computa projeções e constrói de forma sistemática modelos de FCFF (Enterprise Value) e FCFE (Equity Value) a partir de premissas dinâmicas do usuário, gerando matrizes de sensibilidade multidimensionais para determinação do valor justo por ação.
